### 🚀 **Installation Steps**

<div align="center">

<table>
<tr>
<td align="center" width="100%">

```bash
# 📦 Clone the repository
git clone https://github.com/SakshL/React.jsPortfolio.git

# 📂 Navigate to project
cd React.jsPortfolio

# ⚡ Install dependencies (super fast with pnpm!)
pnpm install

# 🔧 Configure environment
cp .env.example .env.local

# 🚀 Start development server
pnpm dev
```

</tr>
</table>

## 🔐 **ENVIRONMENT SETUP**

<img src="https://user-images.githubusercontent.com/74038190/212257465-7ce8d493-cac5-494e-982a-5a9deb852c4b.gif" width="100">

</div>

Create your `.env.local` file:

```env
# 🔑 GitHub Integration (Required)
GITHUB_ACCESS_TOKEN=ghp_your_token_here

# 📧 Contact Form (Required)  
BREVO_API_KEY=xkeysib-your-api-key-here

# 📊 Analytics (Optional)
HOTJAR_ID=your_hotjar_id
```

## � **Quick Start**

### 📋 **Prerequisites**
```bash
Node.js >= 18.0.0
pnpm >= 8.0.0 (recommended)
```

### ⚡ **Installation**

```bash
# Clone this awesome repository
git clone https://github.com/SakshL/React.jsPortfolio.git

# Navigate to project directory
cd React.jsPortfolio

# Install dependencies with pnpm (faster than npm!)
pnpm install

# Set up environment variables
cp .env.example .env.local
# Add your API keys to .env.local

# Start the development server
pnpm dev
```

<div align="center">

**🎉 Open [http://localhost:3000](http://localhost:3000) and see the magic!**

</div>

## 🔧 **Configuration**

### 🌍 **Environment Variables**

Create a `.env.local` file in the root directory:

```env
# 🔑 Required for GitHub integration
GITHUB_ACCESS_TOKEN=your_github_token_here

# 📧 Required for contact form
BREVO_API_KEY=your_brevo_api_key_here

# 📊 Optional for analytics
HOTJAR_ID=your_hotjar_id_here
```
